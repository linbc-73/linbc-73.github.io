论文笔记：G2GT Retrosynthesis Prediction with Graph-to-Graph Attention Neural Network and Self-Training

发表于2023-04-17|更新于2023-04-19
|阅读量:

Basic Information

Title: G2GT Retrosynthesis Prediction with Graph-to-Graph Attention Neural Network and Self-Training
Authors: Zaiyun Lin, Shiqiu Yin, Lei Shi, Wenbiao Zhou, YingSheng Zhang (Beijing Stonewise Technology)
Codes: https://github.com/Anonnoname/G2GT_2
Cite This: J. Chem. Inf. Model. 2023, 63, 1894−1905

Abstract & Conclusion

Abstract：

针对retrosynthesis（逆合成）的一个graph to graph transformation model(G2GT)，此外data augmentation进行self training可以提升模型的性能，以及提出了一个weak ensemble的inference策略可以提升生成的多样性。
实验的结果在USPTO-50K和USPTO-Full数据集上top-1 accuracy等指标表现达到SOTA。

  • 思考：之前做template free的一般都是使用smiles作为输入输出，用graph数据（尤其是输出）会带来什么挑战？以及有什么优势？

Conclusion：

模型的graph decoder基于Transformer实现，可以支持训练阶段每个time step进行parallel prediction并行预测。
USPTO-FULL是一个更接近现实的数据集，它没有可靠的atom-mapping, 没有prior reaction class 和 matching templates信息。
G2GT目前的局限是top-10的表现一般，对于大分子self-attention的平方复杂度对效率有影响。

Introduction

目前主流的retrosynthesis方法可分为三种，分别是template-based, template-free 和 semi-template 的途径。
1）template-base method：根据一些graph transformation template将生成物映射为反应物，这些template主要来自于专家知识或者machine learning的方法生成，后者需要用到atom mapping的知识（这本身就是一个仍然有挑战的问题）。
2）semi-template method：将retrosynthesis分为两步，第一步是预测产物的反应中心，第二步是根据反应中心将产物分解为synthons（合成子），通过合成子转化为生成物。这种做法的缺陷是依赖atom mapping技术以及反应种类受限。
3）template-free method：将retrosynthesis视为Seq2Seq的翻译工作，这种方法可以泛化到尚未发现的反应上。人们尝试使用LSTM和AT（Augmented Transformer）作为backbone进行实验。
本文采用template-free的方法，并且使用graph作为输入，因为graph比SMILES更符合分子的自然形式，也对分子顺序没有要求。贡献介绍同Abstract节。

Method

G2GT framework

G2GT模型的encoder是Graphormer，将输入的graph转化为原子个数|V|个隐向量的集合{z}。框架如论文Figure1所示：

由于不能预先知道生成的graph的结构信息，不能直接使用Graphormer的框架作为decoder，作者设计了新的decoder结构。
首先，定义decoder生成的序列，Y = {v1,v2, e2, 2−1,···,vn, en,n−1, …, en,1}，其中vi为第i个原子，ei-j为第i和第j个原子之间的连边（注意边的顺序是倒序），规定，如果节点j与编号小于等于i的节点(i < j)都没有连边，则去掉尾部ej-i到ej-1的null边，如果节点j与节点k、节点i（i < k < j）有连边，但节点j与节点i+1, i+2, … k-1都没有连边，则将ej-(k-1), … ej-(i+1)换为GapN，其中N等于k-i-1（边数），例子如论文Figure3所示。
Q： 对于ground truth data这个定义是明确的，但是inference的时候如何确定哪个边是有效的及其顺序呢？
可以将生成任务的每一步看做一次classification，一共有atoms × types of chirality + number of bonds个类别，loss function是cross-entropy。

decoder的框架如论文Figure2所示。不同于Graphormer在attention layer添加的三种结构信息bias，G2GT decoder使用一个新定义的Centrality和Laplacian Positional Encoding信息。

Centrality原先的设计只能适用于已知了整个graph结构的情况。为了避免information leakage的问题，在生成到第y步的时候，central matrix为yydimension维，如论文Figure4所示，central matrix[j][i]表示生成到第j步时，元素i的centrality，如果元素i为一个边元素，则该值为0，若元素i为节点，则表示节点i与当前已经生成的所有节点的连变数。

Laplacian Positional Encoding(LPE)
PE是不同频率的 sine 和 cosine 函数，LPE的定义不太理解，需要学习一下ref[23]。

Data Augmentation & Decoding Strategy

由于模型框架是graph permutation invariant的，不能使用一般SMILES所用的augmentation方法。文中使用了类似AlphaFold的self-training method，先训练一个G2GT模型，然后用该模型在training data以外的数据集上生成，使用置信度较高的生成结果进行进一步的训练。
对于同一种产物，可能的retrosynthesis路径有很多种，为了提升生成结果的diversity，这篇工作采用多种inference策略结合进行decoding，包括Beam Search, Top-p, 和 Top-k。Top-k方法可以减少Top-p在概率分布过于平均时的采样数量，提高效率，同时使用了更高的temperature以提高diversity。对于同一个产物，一共sample m次以获得top-n result（n < m）。
而这篇工作还使用了类似ensemble learning的weak ensemble的方法，给每个产物加上一个tagid（tagid和reaction类型是什么关系？没太看懂），作者对该方法进行了Ablation Study。
